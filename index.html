کد ها to do ...
git config وارد کردن اطلاعات فولدر و برنامه نویس
git config --global user.name"example" وارد کردن اسم
git config --global user.email"example@email.com" وارد کردن ایمیل
git config --list نمایش دادن اطلاعات وارد شده
git touch "file" ساختن فایل
git status وضعیت فایل ها
git add "file" staged اضافه کردن فایل به
git commit -m"insert add what to app" اضافه شدن زمان و مکان و فرد با ارائه توضیحات
